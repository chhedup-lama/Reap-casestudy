Platform-First Design Principle

💡
Principle: Build every KYB capability as a versioned API service, not a hard-wired feature. KYB is the first consumer, not the only one.

Service API Contracts

[ Each platform service exposes a stable, versioned REST API. KYB-specific logic lives in the product layer, not the platform service. ]
